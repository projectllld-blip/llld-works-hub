# Security Policy

LLLD Works Hub / Works Market ローカル検証版のセキュリティ方針。

## 基本方針

- GitHub Pagesは静的公開であり、置いたHTML、JS、画像、JSONはURLを知っていれば見られる前提で扱う。
- APIキー、秘密情報、個人情報、有料商品の実ファイルは置かない。
- ログイン、決済、購入履歴、投稿審査、権限管理はフロントだけで実装しない。
- 外部投稿は将来的にもまず審査制にする。

## 置いてよいもの

- 公開してよいHTML、CSS、JavaScript
- 公開してよいサムネイル、説明画像
- 公開してよいサンプルデータ
- 商品説明、価格、販売状態、問い合わせ導線
- 無料公開するアプリ本体
- 公開して問題ないMarkdown資料

## 置いてはいけないもの

- APIキー、アクセストークン、秘密鍵
- 本番アカウント情報
- 顧客名簿、生徒情報、スタッフ個人情報
- 勤怠の実データ
- 教材PDF、入試問題PDF、著作権上公開できない資料
- 契約書、見積書、請求書、財務資料
- 有料商品の実ファイル
- 購入履歴、売上情報、審査状況、権限情報

## 実装ルール

- APIキーが必要な処理は、将来Supabase Edge Functions等のサーバー側に置く。
- フロントHTML/JSに秘密情報を書かない。
- 有料商品の納品は、メール添付、Google Drive限定リンク、手動共有で行う。
- `data/` は仮DBであり、公開されても問題ない情報だけを置く。
