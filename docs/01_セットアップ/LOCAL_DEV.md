# LLLD Works Hub ローカル検証版

このフォルダは、本番GitHub Pagesを壊さずにマーケット化を検証するためのローカルコピーです。

## 目的

- 本番リポジトリへ直接実装しない
- まずローカルで動作確認する
- 問題がない変更だけ、本番リポジトリへ移す
- GitHubへのpushは本番反映時だけ行う

## 起動方法

```bash
cd "/Users/tetsuya/川岡哲也専用　AI軍団/04_社内運営/02_案件管理/コンテンツ販売（ローカル検証版）"
python3 -m http.server 5500
```

確認URL:

```text
http://localhost:5500/
```

## 作業ルール

- この検証版では自由に試してよい
- HTMLを直接ダブルクリックせず、必ずlocalhostで確認する
- 変更が固まるまではGitHubにpushしない
- 本番へ移す前に、変更ファイルと確認結果を整理する
- 本番へ反映するときは、本番リポジトリ側で必要な差分だけを移植する

## 本番へ移す前の確認

- トップページが表示される
- CSS / JS / JSON が読み込める
- 既存コンテンツへのリンクが壊れていない
- 最終利用履歴8件の思想が残っている
- localStorageへ重要情報を保存していない
- GitHub Pagesでも動く相対パスになっている
