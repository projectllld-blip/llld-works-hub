# Admin Console Skeleton

LLLD Works Market 管理画面の骨組みメモ。

## 現在の位置づけ

`admin.html` はローカル検証専用のモック画面。
本番操作はできない。

## 想定する管理対象

- コンテンツ
- 投稿者
- カテゴリ
- 投稿申請
- 問い合わせ
- 購入相談
- 手動納品
- 公開前監査

## ロール

- guest: 公開Marketを見るだけ
- member: 将来の会員
- author: 投稿者
- admin: 管理者

現時点では `AuthMockService` の表示確認用ロールであり、本番権限ではない。

## 管理画面で将来必要な操作

- 投稿申請の受付
- 著作権チェック
- 個人情報チェック
- 公開可否判断
- 販売状態の更新
- 手動納品状況の記録
- 問い合わせ対応状況の記録

## 本番化するときの条件

- Supabase等のDBがある。
- RLSが設定されている。
- 管理者認証がある。
- 操作ログを残せる。
- 秘密キーがフロントに出ない。
- GitHub Pagesだけで完結させない。
