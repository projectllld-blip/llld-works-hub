# Phase 6.1 管理画面モック改善記録

作成日: 2026-06-25

## 目的

Phase 6 Skeleton の `admin.html` を、審査・公開判断・販売状態確認ができる管理画面モックへ改善する。

## 追加した確認機能

- コンテンツ一覧の行アクション
  - 詳細
  - 審査
  - 編集モック
  - 公開判定
- 詳細パネル
- 審査チェックリスト
- 公開判定パネル
- 投稿申請一覧モック

## 安全方針

- 本番DB接続なし。
- 認証なし。
- 決済なし。
- 実データ書き込みなし。
- localStorage保存なし。ただし既存の表示確認用モックロールのみ例外。
- 公開状態、審査状態、販売状態は画面上の確認用で、`data/contents.json` には反映しない。

## 次に進める条件

- 投稿申請、販売管理、審査管理を本当にDBへ持たせるか判断する。
- Supabase開発環境を作る。
- RLS、管理者権限、操作ログの方針を決める。
- 本番反映前に `docs/02_監査/release-checklist.md` を通す。
