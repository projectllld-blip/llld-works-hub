# Phase 6.2 商品詳細・販売導線強化記録

作成日: 2026-06-25

## 目的

ユーザー側の販売導線を強化し、各コンテンツについて「誰向けで、何ができて、どう使う/相談するのか」が分かる状態にする。

## 変更内容

- `content-detail.html` の表示を `assets/js/marketPages.js` 側で整理。
- 詳細ページで公開状態、販売状態、提供方法、対象ユーザー、CTA種別を確認できるようにした。
- 解決する悩み、主な機能、内容物、使い方、注意事項をデータがある場合だけ自然に表示するようにした。
- CTAルールを `priceType` と `saleStatus` に沿って整理した。
- `request.html` で相談対象コンテンツの概要を表示できるようにした。

## 実装しないこと

- 決済連携
- 認証
- DB接続
- 自動納品
- 購入履歴保存
- 個人情報保存

## 確認観点

- Marketから詳細へ移動できる。
- 詳細から無料利用、購入相談、開発相談、先行案内へ進める。
- 有料商品の本体ファイルをGitHub Pagesに置いていない。
- requestページで相談対象が分かる。
- 既存Hub導線を壊していない。
