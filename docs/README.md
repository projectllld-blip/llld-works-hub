# docs フォルダ

このフォルダは、LLLD Works Hub / Market ローカル検証版のルール、構想、監査、フェーズ記録を置く場所です。

## 区分

```text
docs/
├── 00_パッケージ構造/  # フォルダ構造、渡すときの見取り図
├── 01_セットアップ/    # ローカル起動、検証手順
├── 02_監査/            # 毎回確認するチェックリスト
├── 03_構想/            # サービス構想、ゴール、進行方針
├── 04_フェーズ記録/    # Phaseごとの記録
├── 05_実装指示/        # Codex等に渡した実装プロンプト
└── context/            # 現在の作業文脈
```

実行に必要なHTML、CSS、JS、JSONは原則として `docs/` には置きません。
