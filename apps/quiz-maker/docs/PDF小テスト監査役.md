# PDF小テスト監査役

## 目的

PDF小テストを追加・修正するたびに、次の4点を必ず確認する。

- 問題と答えが同じ回・同じ単元に対応している
- 問題選択欄に単元名・見出しが表示される
- PDF画像の向きが問題本文の向きと合っている
- 掲載しない教材が manifest に入っていない

## 監査役の役割

### 1. 対応関係監査

問題PDFと解答PDFを同時に開き、見出し・回番号・内容が一致しているか確認する。

- 第1回の問題に第4回の解答が入っていないか
- まとめ回や飛び番の解答を、単純な連番で割り当てていないか
- 1ページに複数回の解答がある場合、見た目上の配置で切り出しているか
- PDF内部座標ではなく、実際に表示される向きを基準に判断しているか

### 2. 単元名監査

manifest の `unitTitle` を確認し、問題選択欄に意味のある見出しが出る状態にする。

- `第1回` だけで終わっていないか
- 空欄、`単元`、`第1回` のような回番号だけの `unitTitle` は不合格として扱う
- 中学受験新演習 社会は、全学年・全回で問題ページ上部の単元名を `unitTitle` に記載する
- 単元名の記載がない場合は、問題PDFの見出しを読み取り、公開前に必ず `unitTitle` へ追記する
- 問題ページ上部の見出しと `unitTitle` が一致しているか
- 読み取れない場合は、無理に推測せず要確認として残す
- UI上で「回番号」と「単元名・見出し」が分かれて見えるか

### 3. 画像向き監査

問題・解答をサンプル表示し、読める向きになっているか確認する。

- 標準新演習は特に横倒しになりやすい
- `/Rotate` メタデータに頼った切り出しで、見た目が回転していないか
- 問題と解答で向きが違っていないか
- 印刷プレビューでもA4内に収まっているか

### 4. 掲載可否監査

公開対象に入れる教材だけが manifest に残っているか確認する。

- 今回、定期テスト対策ワークは取り下げ扱いのため掲載しない
- 掲載しない教材はPDFファイルが残っていても、`manifest.json` と `pdf_quiz_manifest.js` から外す
- グループ名・教材名・教科で絞り込み、UIに出てこないことを確認する

## 毎回の確認手順

1. `manifest.json` の件数、グループ別件数、PDF参照切れを確認する。
2. 各教材から第1回・中間回・最終回を選び、問題と解答を画像で確認する。
3. 解答ページが複数回収録の場合は、全ての切り出し位置を見た目基準で確認する。
4. 問題選択欄に単元名・見出しが表示されることをブラウザで確認する。
5. 非掲載教材がUIに出ないことを確認する。
6. 確認結果と残リスクを作業メモに残す。

## 2026-06-24 修正での決定

- 中学受験新演習 社会の解答切り出しは、見た目上の配置を基準にする。
- 中学受験新演習 社会の4分割解答ページは、右上、左上、右下、左下の順で第1回、第2回、第3回、第4回に対応する。
- 中学受験新演習 社会は、問題選択欄に各回の単元名を表示する。`第n回` だけの表示に戻さない。
- 標準新演習の4分割解答ページは、左上、左下、右上、右下の順で第1回、第2回、第3回、第4回に対応する。
- 標準新演習は原本表示の向きをそのまま使い、PDF内部の回転指定で追加回転しない。
- 定期テスト対策ワークは、今回は掲載しない。

## 完了条件

- `manifest.json` と `pdf_quiz_manifest.js` のPDF参照切れが0件
- UIで定期テスト対策ワークが表示されない
- 中学受験新演習 社会と標準新演習 社会の問題・解答サンプルが対応している
- 中学受験新演習 社会の `unitTitle` が、空欄または `第n回` だけになっていない
- 標準新演習の問題・解答が横倒しになっていない
- 変更内容をGitHubへcommit/pushしている
